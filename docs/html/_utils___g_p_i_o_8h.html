◆ GPIOSetupSimple()

int GPIOSetupSimple ( Uint32 Pin,
Uint32 Direction )

Quick setup of a simple input/output pin.

This function configures a simple input/output pin with the following assumptions:

  • It uses the first peripheral (GPIO).
  • It disables the pull up resistor.
  • It syncronizes with SYSCLKOUT.
  • It has a sampling qualification period of one period of SYSCLKOUT.
